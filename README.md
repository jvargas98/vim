# Vim commands

## Modes

- ### Insert mode
    `i` - *Enter to insert mode.*
- ### Normal mode
    `Esc` - *Enter to normal mode.*

## Basic movements

- `h` - Move to the left.
- `l` - Move to the right.
- `k` - Move to the up.
- `j` - Move to the down.

## Word movements

- `b` - Move to the beginning of the word.
- `w` - Move to the start of the next word.
- `e` - Move to the end of the word.
